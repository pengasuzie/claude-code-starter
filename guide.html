Welcome — let's get you set up.

This is your marketing toolkit — it turns Claude into a seriously good SEO expert, marketer and brand guardian with 42 skills covering copywriting, SEO, CRO, strategy, analytics, and more. Plus live data from Google Search Console and GA4. You just talk to it — right inside VS Code.

💡 The Big Idea

Before we do anything, here's the one concept that makes everything click.

You describe each client once. Every skill remembers.

Inside this project is a hidden folder called .claude/. It's like Claude's brain for this project. When you tell Claude about a client — their audience, voice, goals — it saves that info. Then every time you ask it to write copy, audit SEO, or plan content, it already knows who it's working for. No repeating yourself.

💬
Quick heads-up: This uses Claude Code inside VS Code (not the Claude chat app). Same Claude brain, but this version can read files, remember context between tasks, and work directly in your editor. That's what makes the magic work.

⚙️ Setup

Five quick steps and you're good to go. Takes about 10 minutes.

1

Install VS Code

VS Code is a free code editor from Microsoft — it's where everything runs. Go to code.visualstudio.com and click the big blue "Download" button. It'll detect whether you're on Mac or Windows automatically.

  • Mac: Open the downloaded .zip file. Drag the "Visual Studio Code" app into your Applications folder. Then open it from Applications (or search "Visual Studio Code" in Spotlight).
  • Windows: Run the downloaded .exe installer. Click "Next" through each step, tick "Add to PATH" if it asks, then click "Install". Open it from the Start menu when it's done.
2

Grab Node.js

Claude Code needs this to run behind the scenes. Head to nodejs.org, click the big green "LTS" download button, open the downloaded file and click through the installer (just keep hitting "Continue" / "Next"). You won't need to think about it again.

3

Install the Claude Code extension

Open VS Code. On the left sidebar, click the Extensions icon (it looks like four small squares). In the search box at the top, type "Claude Code". Find the one published by Anthropic (it'll have a blue tick and 2M+ installs) and click Install. Double-check the publisher says "Anthropic" — there are copycats.

4

Sign in

You need a Claude Pro, Max, Team, or Enterprise subscription. After installing the extension, it'll prompt you to sign in with your Anthropic account. Follow the login flow and you're done.

💰
Pay-as-you-go option: Instead of a subscription, you can use API credits. Sign up at console.anthropic.com, click "API Keys" in the sidebar, create a key, and add credits. Claude Code will ask for the key on first launch.

📋 Client Briefs

This is the secret sauce. Set up a brief once, and every skill becomes client-aware.

How it works

Run /client-brief and Claude walks you through a friendly interview — who's the client, what do they do, who's their audience, what's their voice like? It saves everything to a context file. After that, whenever you ask for copy, strategy, or an audit, Claude already knows the client. No repeating yourself.

What Claude knows after a brief

  • Target audience and their pain points
  • Brand voice (e.g. "confident but not arrogant")
  • SEO keywords and content priorities
  • Competitors and market positioning
  • Products, services, and differentiators

Multiple clients? No problem.

Each client gets their own brief file in .claude/clients/. Switch clients by saying "I'm working on Sunrise Bakery" and Claude loads the right context.

🎯 Pro Tips

A few things that make a real difference in quality.

Be specific, not vague

Meh: "Write a blog post about marketing"
Great: "Write a 1,200-word blog post for CFOs about why fractional CMOs outperform agencies, confident tone, include data"

Share URLs

Claude can read live web pages. Paste a competitor's site, a client landing page, or an article you like — it'll use it as context.

Iterate, don't settle

First drafts are starting points. Say things like: "Make the headline punchier", "Too formal — make it conversational", "Add more examples", "Cut this by 30%".

Feed it reference material

Got keyword research, analytics screenshots, brand guidelines, or competitor copy? Share them. More context = better output.
