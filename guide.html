Welcome to Claude Code Starter

This project comes pre-configured with 20 skills covering voice/audio, strategic analysis, opportunity evaluation, and product development. Just talk to Claude naturally and it picks the right skill.

⚙ Setup

Three steps to get running.

1

Install Claude Code

If you haven't already, install Claude Code globally:

Terminal
$ npm install -g @anthropic-ai/claude-code

You'll need an Anthropic API key.

2

Open this project

Navigate to the project folder and start Claude Code:

Terminal
$ cd claude-code-starter
$ claude
3

Start talking

Claude automatically loads all skills from the .claude/ folder. Just ask for what you need in plain language.

💡
Permissions: The included settings.local.json pre-approves common tools (file editing, web search, bash commands) so you won't get permission prompts for every action.

🎯 All Skills

20 skills organised into four categories. Click any skill to see what it does.

Voice & Audio
/voice-to-text Transcribe audio or video to text Voice

Transcribes audio and video files to text with speaker identification. Supports common formats (MP3, WAV, M4A, MP4, etc.). Outputs a clean transcript with timestamps and speaker labels.

  • Automatic speaker diarisation
  • Handles long recordings
  • Outputs structured text files

Try: "Transcribe this meeting recording"

/text-to-voice Convert text to spoken audio Voice

Converts any text file to an MP3 audio file using local text-to-speech. Supports three engines: Kokoro, Orpheus, and Coqui XTTS v2.

  • Multiple voice options
  • Runs locally — no cloud API needed
  • Good for turning documents into listenable content

Try: "Turn this document into audio"

/voice-briefing Create a spoken audio briefing Voice

Takes a meeting transcript or document and converts it into a concise spoken briefing. Summarises the key points, then generates audio you can listen to on the go.

  • Summarises before converting to audio
  • Focuses on decisions, action items, and key themes
  • Great for catching up on meetings you missed

Try: "Turn this meeting transcript into an audio briefing"

📊
Strategic Analysis
/strategic-analysis Analyse documents with context Strategy

Analyses meetings, documents, or transcripts with full project context awareness. Produces strategic analysis grounded in actual capabilities, roadmap, and competitive landscape.

  • Identifies key themes and strategic implications
  • Highlights competitive threats and opportunities
  • Produces clear action items

Try: "Analyse this transcript and focus on competitive threats"

/competitive-intel Competitive gap analysis Strategy

Runs a structured competitive gap analysis — classifying signals vs noise, building a gap matrix, and producing an actionable report.

  • Signal vs noise classification
  • Competitive gap matrix
  • Prioritised recommendations

Try: "Run a competitive analysis on our market"

💡
Opportunity Evaluation
/opportunity-brief Capture a business opportunity Eval

Walks you through capturing a new opportunity in a structured format — the pitch, business model, key people, initial risks, and why now.

Try: "I have a business idea — help me evaluate it"

/market-research Size the market, find signals Eval

Researches the market for an opportunity — sizing the market, identifying demand signals, checking trends, and finding relevant data points.

Try: "How big is the [X] market?"

/competitor-landscape Map the existing players Eval

Maps out who else is operating in a market. Identifies direct competitors, adjacent players, gaps, and opportunities for differentiation.

Try: "Who are the main players in [market]?"

/strategy-session Brainstorm go-to-market Eval

A creative brainstorming session for go-to-market strategy, differentiation, blue ocean thinking, and positioning.

Try: "Let's brainstorm how to position this"

/due-diligence Vet the opportunity Eval

Vets the people, financials, legal, and technical feasibility of an opportunity before you commit time or money.

Try: "Run due diligence on this opportunity"

/viability-assessment Score and decide Eval

Scores an opportunity across 8 weighted dimensions and gives you a clear go/no-go recommendation with reasoning.

  • 8.0+ — Strong Go
  • 6.5–7.9 — Conditional Go
  • 5.0–6.4 — Proceed with caution
  • 3.5–4.9 — Likely No
  • Below 3.5 — Hard No

Try: "Score this opportunity — should I pursue it?"

Product Development (BMAD)
/bmad Structured product development Product

The BMAD Method gives you specialised AI agents for each stage of product development:

  • Business Analyst — Product briefs and market context
  • Product Manager — PRDs and technical specs

  • UX Designer — User experience and interaction design

Try: "Help me write a PRD for [product]"

💭 Tips for Best Results

A few things that make Claude work better for you.

Be specific

  • Instead of: "Analyse this"
  • Say: "Analyse this meeting transcript and focus on competitive threats and action items for the engineering team"

Give it context

  • Share URLs and let Claude read them
  • Paste in documents you want analysed
  • Share competitor URLs for landscape mapping

Use web search

Claude can search the web in real time:

  • "Search for [competitor] and analyse their positioning"
  • "What are people on Reddit saying about [topic]?"
  • "Find the latest stats on [industry trend]"

Useful commands

  • /clear — Clear conversation history
  • /compact — Free memory without losing context
  • /help — Get help with Claude Code
  • Ctrl+C or /exit — Exit Claude Code
⚡
Long conversations use memory. If Claude starts getting slower or forgetting things, use /compact to compress the conversation without losing context. Or use /clear to start fresh.

✏ Customise

Everything is editable. Make it yours.

Edit a skill

Open .claude/skills/[name]/SKILL.md in any text editor. Skills are just markdown files that teach Claude how to approach specific tasks.

Add a new skill

Create a new folder in .claude/skills/ with a SKILL.md file. Or browse and install from the open skills ecosystem:

Terminal
$ npx skills find seo
$ npx skills find cloudflare
$ npx skills find react

Browse available skills at skills.sh

Edit project instructions

Open .claude/CLAUDE.md — this is what Claude reads when it starts. Add context about your projects, preferences, and conventions.

Add a command

Create a new .md file in .claude/commands/ to add a custom slash command.
